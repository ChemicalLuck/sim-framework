# @chemicalluck/create-sim-game

Scaffold a new [**sim**](https://github.com/ChemicalLuck/sim-framework) game.

```bash
npm create @chemicalluck/sim-game my-game
cd my-game
npm install
npm run dev        # game at /, content editor at /editor
```

This creates a minimal, complete game — a couple of locations, an item, and all the
required content files — wired to `@chemicalluck/sim-engine` and the `sim` CLI. Edit
`src/game/data/*.json` to build your world.
